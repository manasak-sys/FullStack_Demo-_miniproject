# Fullstack Demo Application

## Description
The Fullstack Demo Application is a complete web application that demonstrates frontend and backend integration using APIs. The backend is developed using Django REST Framework, where items are added and managed through the admin panel and exposed via APIs. The frontend is built using React and JavaScript to consume these APIs and display data dynamically.

## Technologies Used
- Python
- Django
- Django REST Framework
- REST APIs
- React
- JavaScript
- HTML
- CSS
- SQLite

## Features
- Backend API for managing items
- Admin panel to add and update items
- RESTful API integration
- React-based frontend to fetch and display data
- Dynamic UI updates using JavaScript
- Fullstack architecture (Frontend + Backend)

## Workflow
1. Admin adds items in the backend admin panel.
2. Backend exposes item data through REST APIs.
3. Frontend consumes APIs using HTTP requests.
4. Items are displayed dynamically in the React frontend.

## Project Structure
- Backend: Django REST API
- Frontend: React with JavaScript
- API-based communication between frontend and backend

## How to Run the Project
### Backend
1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Run migrations using `python manage.py migrate`
4. Create superuser using `python manage.py createsuperuser`
5. Start backend server using `python manage.py runserver`

### Frontend
1. Navigate to frontend directory
2. Install dependencies using `npm install`
3. Start frontend server using `npm start`
